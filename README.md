#Steps:
Prereq: download python3

## Windows:

1. Open up your command line
2. Run pip install fastapi
3. Run pip install uvicorn
4. To start the app, run 'python -m uvicorn server:app --reload'
5. Navigate to http://127.0.0.1:8000/docs

## Macs:

1. Open up your command line
2. Run pip install fastapi
3. Run pip install uvicorn
4. To start the app, run 'python -m uvicorn server:app --reload'
5. Navigate to http://127.0.0.1:8000/docs
